# Machine Learning 

This repository is a comprehensive collection of machine learning algorithms and projects implemented during my **3rd Year, 6th Semester** B.Tech in Computer Science and Engineering. 

It serves as both a record of my academic coursework and a living lab where I continue to hone my skills in data science and predictive modeling.

## 📌 Projects & Algorithms Implemented
The notebooks in this repository cover a wide range of supervised and unsupervised learning techniques:

* **Regression:** Logistic Regression, Multiple Linear Regression, and Random Forest Regressor.
* **Classification:** Decision Trees, Naive Bayes, and K-Nearest Neighbors (KNN).
* **Clustering:** K-Means Clustering for unsupervised data grouping.
* **Advanced Ensembles:** Random Forest Classification for high-accuracy predictions.
* **Industrial Applications:** Industrial Regression models focusing on real-world datasets.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
* **Environment:** Jupyter Notebook
